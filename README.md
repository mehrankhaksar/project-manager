# 📁 Project Manager

A simple and clean project & task management application built with React, TypeScript, Context API, and Tailwind CSS.

## 🚀 Features

- Create and manage multiple projects
- Add and remove tasks for each project
- Select and view project details
- Global state management using Context API + useReducer
- Reusable UI components (Button, Input, Modal)
- Form validation with user feedback
- Modal system using React Portal
- Fully typed with TypeScript
